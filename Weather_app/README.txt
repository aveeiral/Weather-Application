This is a weather application that uses various libraries like :
1) json
2) requests
3) tkinter
4) pyinstaller

The API is generated from the website : https://openweathermap.org

To run the application just open the "Weather Application". 
Now you can enter your city and know the current weather of your city.
This is just a simple approach of having a executable application.
You can find and edit the code in the folder...
